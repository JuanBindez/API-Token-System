API Documentation

Introduction

This API allows you to generate a token associated with a username and access protected data using that token. Below, you will find details on how to use this API.

Generate Token

To generate a token, follow these steps:

  1. Go to the main page.
  2. Enter your username and click on "Generate Token".
  3. The generated token will be displayed on the screen.

Example request using requests:

import requests

response = requests.post('http://127.0.0.1:5000/', data={'username': 'your_username'})
print(response.text)

Access Protected Data

To access the protected data, send a GET request to the URL /api/data with the generated token in the authorization header.

Example request using requests:

import requests

url = 'http://127.0.0.1:5000/api/data'
headers = {'Authorization': 'your_token_here'}
response = requests.get(url, headers=headers)

print(response.json())

If the token is valid, you will receive a response containing the protected data.

List of Endpoints

  • GET /api/data: Returns protected data, requires the Authorization header with a valid token.
